# Platform Requirements

**Last Updated:** November 4, 2025

---

To ensure a smooth trading experience with MarketEdgePros, please make sure your computer meets the minimum requirements for running the MetaTrader 5 (MT5) platform.

## Minimum System Requirements

These are the absolute minimum requirements to run MT5. Performance may be slow.

- **Operating System:** Windows 7/8/10; macOS Mojave (10.14) or higher
- **Processor:** 2.0 GHz or faster
- **RAM:** 512 MB
- **Screen Resolution:** 1024x768
- **Internet:** Broadband internet connection


## Recommended System Requirements

For optimal performance, especially if you plan to use multiple charts or Expert Advisors, we recommend the following:

- **Operating System:** Windows 10/11 (64-bit); macOS Monterey (12.0) or higher
- **Processor:** Multi-core processor, 3.0 GHz or faster
- **RAM:** 8 GB or more
- **Hard Drive:** Solid State Drive (SSD)
- **Screen Resolution:** 1920x1080 or higher
- **Internet:** Stable, low-latency broadband connection (wired Ethernet is recommended over Wi-Fi).


## Mobile Trading

MT5 is also available for mobile devices.

### iOS
- **Device:** iPhone or iPad
- **Operating System:** iOS 11.0 or later

### Android
- **Device:** Smartphone or tablet
- **Operating System:** Android 5.0 or later


## Linux Users

While there is no native MT5 application for Linux, it can be run using Wine. Please refer to the official MQL5 community guide for instructions on how to install it on popular Linux distributions like Ubuntu.

- [Installing MetaTrader 5 on Linux](https://www.mql5.com/en/articles/200)
